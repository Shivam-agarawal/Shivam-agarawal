## 🚀 Featured Projects

### 💬 [Whisprr](https://github.com/Shivam-agarawal/Whisprr) — Real-Time Chat Application
A full-stack real-time messaging app built with the MERN stack and Socket.io, developed during the **Udaan** intercollege hackathon. Supports instant messaging, image sharing via Cloudinary, live online-status indicators, and secure authentication.

- 🔐 JWT authentication stored in httpOnly cookies with bcrypt password hashing
- 🛡️ Arcjet middleware for bot detection, rate limiting, and SQLi/XSS protection
- ⚡ Real-time delivery via Socket.io with optimistic UI updates
- 🗂️ Zustand for lightweight global state management

**Tech:** React, Node.js, Express, MongoDB, Socket.io, Cloudinary, Zustand, TailwindCSS
**Live Demo:** [whisprr-1ebn.onrender.com](https://whisprr-1ebn.onrender.com) · **Repo:** [GitHub](https://github.com/Shivam-agarawal/Whisprr)

### 💻 [CodeSync](https://github.com/Shivam-agarawal/CodeSync) — Real-Time Collaborative Code Editor
A web-based IDE for pair programming, combining Monaco Editor with Yjs CRDTs for zero-conflict multiplayer editing, plus live remote code execution.

- ⚡ True multiplayer editing with conflict-free sync (Yjs + WebSockets)
- 🏷️ Color-coded remote cursor presence for collaborators
- 🏃 Live code execution across multiple languages with stdin support and terminal output (xterm.js)
- 🐳 Fully containerized with a multi-stage Docker build

**Tech:** React, Vite, Monaco Editor, Yjs, Socket.io, Node.js, Express, Docker
**Live Demo:** [code-sync-orcin.vercel.app](https://code-sync-orcin.vercel.app/) · **Repo:** [GitHub](https://github.com/Shivam-agarawal/CodeSync)

### 📧 [MailGen AI](https://github.com/Shivam-agarawal/A.I.-Cold-Email-Generator) — AI Cold Email Generator
A full-stack AI tool that generates structured cold emails, LinkedIn DMs, and follow-ups using LLaMA 3 via the Groq API, with configurable tone and sales frameworks (AIDA, PAS, BAB).

- 🔒 Secure auth with OTP email verification and a complete forgot-password flow
- 📊 Analytics dashboard with Recharts (activity, tone distribution, framework usage)
- 📥 One-click export to PDF/TXT, with PWA support for installation
- 🛡️ Built-in rate limiting to prevent API abuse

**Tech:** React, Node.js, Express, MongoDB, Groq API (LLaMA 3), JWT, Nodemailer
**Live Demo:** [a-i-cold-email-generator.vercel.app](https://a-i-cold-email-generator.vercel.app/) · **Repo:** [GitHub](https://github.com/Shivam-agarawal/A.I.-Cold-Email-Generator)
